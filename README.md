### 可用配置

| 配置项 | 必填 | 默认值 | 功能/备注 |
|---|----|---|----|
|size| optional |l|按钮大小，可选参数l、 m和s, 分别代表大、中和小|
|disabled| optional |false|按钮失效|
|type| optional |primary|按钮的类型, 可选的参数primary和secondary|
|onClick|optional|-|按钮点击时的回调|
